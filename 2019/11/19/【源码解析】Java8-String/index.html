      hashCodeи®Ўз®—пјҡ

      з”ұжәҗз ҒеҸҜд»ҘзңӢеҮәhashеҖјеҲқе§ӢеҢ–ж—¶жҳҜ0пјҢеҪ“иҺ·еҸ–иҝҮдёҖж¬Ўд№ӢеҗҺhashCodeе°ұдјҡиў«и®Ўз®—еҮәжқҘпјҢз„¶еҗҺиў«зј“еӯҳдәҶдёӢжқҘпјҢйҡҸStringдёҖеқ—дёҚеҶҚеҸҳдәҶпјҢд№ҹжӯЈжҳҜеӣ дёәжӯӨпјҢе–ңж¬ўеҗ§StringдҪңдёәHashMapзҡ„key

      hashCodeи®Ўз®—жңүдёӘй—®йўҳпјҡдёәд»Җд№ҲйңҖиҰҒдёҖдёӘдёӯй—ҙеҸҳйҮҸh пјҹ

      зӯ”пјҡжІЎжңүдёҙж—¶еҸҳйҮҸжҳҜзәҝзЁӢдёҚе®үе…Ёзҡ„пјҢеӣ дёәеӨҡзәҝзЁӢи®ҝй—®дёӢпјҢеҰӮжһңжІЎжңүhпјҢеҪ“дёҖдёӘзәҝзЁӢиҝӣиЎҢи®Ўз®—ж—¶пјҢз¬¬дәҢдёӘзәҝзЁӢиҝӣжқҘпјҢжӯӨж—¶hashдёҚдёә0еҲҷзӣҙжҺҘиҝ”еӣһжӯӨж—¶й”ҷиҜҜзҡ„еҖјгҖӮ

      StringдҝқжҢҒдёҚеҸҜеҸҳжҖ§йҮҮеҸ–дәҶе“ӘдәӣжҺӘж–Ҫпјҹ

      • value[] дҪҝз”Ё private final дҝ®йҘ°
    • жһ„йҖ еҮҪж•°дёӯеӨҚеҲ¶е®һеҸӮзҡ„еҖјз»ҷ value[]
      • дёҚеҜ№еӨ–жҸҗдҫӣд»»дҪ•дҝ®ж”№ value[] еҖјзҡ„ж–№жі•
    • йңҖиҰҒиҝ”еӣһ String зҡ„ж–№жі•пјҢз»қдёҚиҝ”еӣһеҺҹеҜ№иұЎпјҢйғҪжҳҜйҮҚж–° new дёҖдёӘ String иҝ”еӣһ
