  2. 新增一个元素

    可以看到调用的linkLast方法：

    1. 先把最后一个元素（即当前元素）赋值给l，此时l为当前元素的引用，为空说明为第一个元素
    2. new一个新的Node，l作为新增元素的pre，新值为新元素value，next为空
    3. last指向新的元素
    4. 如果是第一个元素，first指向新元素，若不是第一个元素则，用当前元素的next指向新的元素
